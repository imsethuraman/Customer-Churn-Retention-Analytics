# Customer-Churn-Retention-Analytics
The project analyzes customer churn behavior and identifies key drivers impacting retention and revenue using SQL and Power BI.

🎯 Objective

Identify churn drivers
Segment high-risk customers
Provide actionable retention strategies

📁 Dataset Features

Customer demographics
Usage behavior
Support tickets
Payment failures
Tenure & activity

🧠 Key Features Engineered

Usage Group (Low / Medium / High)
Risk Segment
Activity Level
Churn Flag

📊 Key KPIs

Total Users: 2800
Churn Rate: 57.32%
Revenue at Risk: 57.41%
Inactive Customers: 1350

🔍 Key Insights

📉 Low engagement customers show the highest churn
🛠️ High support tickets increase churn risk
👤 Inactive users contribute the majority of churn
💰 Significant revenue is at risk
